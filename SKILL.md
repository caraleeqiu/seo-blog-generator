# SEO Blog Generator (English)

Generate SEO-optimized blog posts for Google ranking.

## Trigger Words
- "seo blog"
- "google seo"
- "english blog"
- "write blog for [URL]"

## Features
- Google E-E-A-T compliant content
- Keyword optimization
- Proper H1/H2/H3 structure
- 2000+ word comprehensive articles
- Product research via Google Search
- Image prompt suggestions

## Usage

```
Write an SEO blog for https://mvland.com
```

## Output
- SEO-optimized title (50-60 chars)
- Meta description (150-160 chars)
- Full article with proper headings
- Image placement suggestions
- Keyword distribution
